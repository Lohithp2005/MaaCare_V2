# MaaCare — Maternal Healthcare Assistance Platform
> 🚧 under development

---

  **Note:** The initial version of MaaCare was developed rapidly for a hackathon. Now Rebuilding with **clean architecture, scalable folder structure, maintainable code practices ,authentication and much smoother overall functioning**..

---
MaaCare is a full-stack maternal healthcare assistance platform designed to provide personalized support, pregnancy tracking, and AI-powered healthcare guidance for expecting mothers.

## Features

- **AI Agentic Chatbot**
  - Uses AI agents to provide personalized pregnancy assistance and answer user queries.

- **Diet Recommendation**
  - Generates personalized pregnancy diet plans based on the user's health profile and nutritional needs.

- **Barcode Product Safety Checker**
  - Scan food, medicines, or cosmetic products to determine whether they are safe during pregnancy.

- **Pregnancy Exercise Assistant**
  - Provides real-time posture detection using your device's camera to guide pregnancy-safe exercises.
 
 - **Personalized Health Profile**
  - Users can provide details such as their trimester and health conditions to receive personalized recommendations and guidance.
